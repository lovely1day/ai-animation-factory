## Prompt اختبار للنظام

استخدم هذا النص في حقل **AI Fullstack Generator** داخل الـ Dashboard:

```
simple todo app with nextjs dashboard and prisma database
```

يمكنك أيضاً اختبار آخر:

```
music learning platform with dashboard and user accounts
```

---

# كيف تتأكد أن المصنع يعمل

## 1️⃣ أرسل الـ Prompt

في المتصفح افتح:

```
http://localhost:3000
```

ثم:

```
AI Fullstack Generator
```

ضع الـ prompt واضغط:

```
Generate Fullstack App
```

---

# 2️⃣ تحقق من إنشاء المشروع

في PowerShell نفذ:

```powershell
dir C:\ALI_WORKSPACE\01_PROJECTS
```

يجب أن ترى مجلد جديد مثل:

```
simple-todo-app-with-nextjs-dashboard-and-prisma-database
```

---

# 3️⃣ افتح المشروع

باستخدام Visual Studio Code:

```powershell
code C:\ALI_WORKSPACE\01_PROJECTS\simple-todo-app-with-nextjs-dashboard-and-prisma-database
```

---

# 4️⃣ تأكد أن المشروع كامل

يجب أن ترى:

```
app
package.json
tailwind.config.ts
tsconfig.json
node_modules
```

وهذا يعني أن:

```
Next.js project created
```

---

# 5️⃣ تشغيل المشروع الذي أنشأه الذكاء

داخل مجلد المشروع الجديد:

```powershell
cd C:\ALI_WORKSPACE\01_PROJECTS\simple-todo-app-with-nextjs-dashboard-and-prisma-database
npm install
npm run dev
```

ثم افتح:

```
http://localhost:3000
```

# اختبار API مباشرة (اختبار محترفين)

يمكنك اختبار API بدون الواجهة:

```powershell
curl -X POST http://localhost:3000/api/ai-fullstack -H "Content-Type: application/json" -d "{\"prompt\":\"test ai app\"}"
```

إذا نجح سيظهر:

```json
{
  "project":"test-ai-app"
}
```

---

# اختبار متقدم (فحص المصنع)

بعد تشغيل المشروع نفذ:

```powershell
npm run build
```

إذا ظهر:

```
Compiled successfully
```

فالمصنع يعمل بشكل صحيح.
