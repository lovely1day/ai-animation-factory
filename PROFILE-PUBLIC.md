# AI Animation Factory

## What It Is

AI Animation Factory is an automated pipeline that generates animated episodes from a single idea. It uses AI to write scripts, generate scene images, compose music, create animations, add subtitles, and assemble everything into a final video — all orchestrated through a queue-based worker system. The platform supports multiple AI providers including Claude, Gemini, and local Ollama models.

## Key Features

- **Idea-to-Video Pipeline** — Enter an idea, get a fully produced animated episode
- **Multi-Stage Generation** — Idea, script, images, music, thumbnail, animation, assembly, subtitles
- **Character DNA System** — Persistent character designs with AI-generated portraits via ComfyUI/SDXL
- **Multi-Provider AI** — Claude Sonnet, Gemini, Ollama (local LLMs) with automatic fallback
- **Creative Pipeline** — Criteria-based AI personas with prompt compression between stages
- **Real-Time Dashboard** — Monitor job progress, view analytics, manage projects
- **Parallel Processing** — Images, music, and thumbnails generate simultaneously
- **Auto-Publish** — Optional automatic publishing when episodes complete

## Target Audience

- Content creators seeking automated animation production
- Studios wanting AI-assisted episode generation
- Developers building AI-powered creative tools
