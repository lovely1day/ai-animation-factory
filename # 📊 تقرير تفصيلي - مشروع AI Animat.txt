### 2. الخلفية (Backend)

| المكون | الحالة | التفاصيل |
|--------|--------|----------|
| **API Server** | ✅ يعمل | Express.js على المنفذ 3002 |
| **نقاط النهاية** | ✅ جاهزة | 5 endpoints |
| **معالجة المشاريع** | ✅ تعمل | 4 مراحل (سكريبت → صور → صوت → فيديو) |
| **تخزين المؤقت** | ✅ يعمل | Map في الذاكرة |

**نقاط النهاية (Endpoints):**
```
GET  /                    - معلومات API
GET  /health              - فحص الحالة
GET  /episodes            - قائمة المشاريع
POST /episodes/create     - إنشاء مشروع جديد
GET  /projects/:id        - تفاصيل مشروع
GET  /projects/:id/results - النتائج النهائية
```

## 🚀 خطوات التشغيل الحالية

### المتطلبات
- Node.js 18+

### الأوامر

```bash

# 2. تشغيل API (Terminal 1)

# يعمل على: http://localhost:3002

# 3. تشغيل Web (Terminal 2)
cd apps/web
pnpm dev
# يعمل على: http://localhost:3000
```
